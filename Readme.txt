-- Mình sẽ tổ chức code như sau:
   + Controller: trong phần Java Resource > src > controller
	=> Trong phần này có thêm mục DAO: Mục này sẽ có nhiệm vụ trung gian giữa controller và CSDL
   + Model: trong phần Java Resource > src > model
   + View: trong phần WebContent > view 
	=> Trong phần này các file JSP sẽ để trực tiếp trong các thư muc user và admin
           Các file js, css, img sẽ để trong assets
